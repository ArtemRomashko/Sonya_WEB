Кухонний мінімум

Посуд, техніка та дрібниці без зайвого — що реально потрібно для готування вдома.

Базовий посуд для готування

Не потрібно одразу купувати все. Ось мінімум, з яким можна приготувати 90% страв.

  • Сковорідка (26–28 см, з антипригарним покриттям) — найуніверсальніша.
  • Каструля (2–3 л) — для супів, круп, варіння.
  • Деко або форма для запікання — якщо є духовка.
  • Миска для змішування — підходить звичайна велика.

Ножі та кухонні інструменти

Один хороший ніж — краще, ніж набір з 10 поганих.

  • Кухарський ніж (20–22 см) — для всього: овочів, м'яса, зелені.
  • Дошка для нарізки — бажано дерев'яна або пластикова (не скляна).
  • Овочечистка, тертка, ополоник, лопатка.
  • Консервний відкривач і ножиці для кухні.

Техніка: що варто мати

Техніка полегшує готування, але не вся є необхідною з першого дня.

  • Чайник електричний — must-have, економить час щодня.
  • Мікрохвильовка — для розігріву і швидкого готування.
  • Занурювальний блендер — для супів-пюре, смузі, соусів.
  • Тостер або вафельниця — за бажанням, але не першочергово.

Догляд за посудом

Хороший посуд служить роками — якщо знати, як за ним доглядати.

  • Антипригарну сковорідку — тільки силіконовою або дерев'яною лопаткою, не металевою.
  • Не мий гарячий посуд холодною водою — деформується від перепаду температур.
  • Ножі — мий вручну, не в посудомийній машині.
  • Заточуй ніж раз на 2–3 місяці — тупий ніж небезпечніший за гострий.
